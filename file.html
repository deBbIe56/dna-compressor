DNACompress

Advanced DNA sequence compression using Burrows-Wheeler Transform with Run-Length Encoding and Huffman Coding algorithms

Compression Algorithm Overview

DNACompress uses a multi-stage approach to efficiently compress DNA sequences. The process combines three powerful algorithms to achieve optimal compression ratios.

Burrows-Wheeler Transform

Rearranges the DNA sequence into runs of similar characters, making it more suitable for compression algorithms. This transformation is reversible and lossless.

Run-Length Encoding

Compresses consecutive repeated characters (like AAA or TTT) into a single character and a count. This dramatically reduces file size for sequences with repeated nucleotides.

Huffman Coding

Creates an optimal binary tree of nucleotides based on their frequency in the sequence. More frequent characters are assigned shorter codes, reducing overall file size.

DNA Sequence Input

Supported formats: FASTA (.fasta), Plain text (.txt), and DNA sequences (.seq)
